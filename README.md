# Restaurant Ordering Web App

This is a front-end restaurant website built as a practical demo project. It focuses on creating a simple ordering experience for small businesses without requiring a full backend system.

Live site:  
https://restaurant-ordering-app-786.vercel.app/

## Overview

The app allows users to browse a menu, add items to a cart, and place an order through WhatsApp. When the user checks out, their selected items are automatically formatted into a message and opened in WhatsApp, making it easy to send the order directly to the business.

The idea behind this project was to build something lightweight but actually usable for a real restaurant.

## Features

- Browse menu items
- Add and remove items from a cart
- View order summary
- Generate a pre-filled WhatsApp order message
- Redirect to WhatsApp for checkout
- Responsive design for mobile and desktop

## Tech Stack

- HTML
- CSS
- JavaScript


## Deployment

The project is deployed using Vercel:  
https://restaurant-ordering-app-786.vercel.app/

## Notes

This is a front-end only project. The ordering system works by using WhatsApp as the communication layer instead of building a custom server.
