# Restaurant WhatsApp Ordering App

A responsive restaurant ordering system that allows users to browse a menu, add items to a cart, and place orders via WhatsApp with a pre-filled message.

## Features
- Interactive menu display
- Add/remove items from cart
- Dynamic cart total calculation
- WhatsApp checkout with pre-filled order details
- Responsive UI

## How It Works
Users add items to their cart. When they click "Order via WhatsApp", the app generates a message containing their order and opens WhatsApp with the pre-filled text ready to send.

## Tech Stack
- HTML
- CSS
- JavaScript

## Run Locally
Open `index.html` in your browser.
